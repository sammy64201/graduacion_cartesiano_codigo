# Practica Codex Python

Proyecto pequeno para aprender a pedir cambios a Codex sin tocar archivos fuera de esta carpeta.

## Como ejecutar

Desde esta carpeta:

```powershell
python main.py
```

## Estructura

```text
practica_codex_python/
  README.md
  main.py
  ejercicios/
    __init__.py
    cadenas.py
    numeros.py
    listas.py
```

## Ideas para practicar con Codex

Puedes pedir cambios como estos:

- Cambia `saludar` para que el saludo sea mas formal.
- Modifica `es_par` para que devuelva texto en vez de `True` o `False`.
- Haz que `duplicar_lista` triplique los numeros.
- Agrega una funcion nueva que calcule el promedio de una lista.
- Agrega pruebas simples para las funciones.

## Reglas de practica

- Pide primero un plan antes de modificar.
- Pide cambios pequenos y revisa el resultado.
- Ejecuta `python main.py` despues de cada cambio.
